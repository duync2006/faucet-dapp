# Vibi Faucet Dapp
An application used to faucet ethers for user.

# Installation and Run App

Clone down this repository via this link: _https://github.com/duync2006/faucet-dapp_

## Installation:

**Step 1:** ```cd faucet-dapp/FE```
**Step 2:** ```npm install```
**Step 3:** ``` cd ../BE```
**Step 4:** ```npm install```

## Run app

**Step 1:** Go to FE directory
**Step 2:** ```npm start```
